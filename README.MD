# Anime Info
Quickly fetch anime information from MyAnimeList using [Jikan.me](https://jikan.me)

Supported Browser(s): Opera

Web technologies used:
- Normalize.css
- JQuery.js
- CSS FlexBox & Grid



Playing around with browser extensions. This project is a WIP.
